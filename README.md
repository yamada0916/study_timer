#study_timer

## 概要

学習分野ごとの勉強時間を管理するWebアプリです。

Python(Flask)を用いて個人で開発しています。

##　作成背景

資格学習を進める中で、分野ごとの学習時間を把握し、継続的な学習状況を管理できる仕組みが欲しいと考え、本アプリの開発を始めました。

また、Python(Flask)を用いたWebアプリケーション開発や、Git・GitHubによるバージョン管理、データ構造を意識した設計などを実践的に学ぶことも目的としています。

## 使用技術

-Python
-Flask
-Git
-Github

## 実装済み

-ホーム画面
-科目ごとの詳細ページ
-GitHubによるソースコード管理
